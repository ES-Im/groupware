# 권한 규칙
- ADMIN은 모든 권한 검사를 통과한다.
- 단, 본인 행위가 반드시 필요한 기능은 예외로 둘 수 있다.
    - 예: 본인 출근기록, 본인 퇴근기록
- DEPT_MANAGER는 같은 부서 대상자에 한해 부서 관리자 권한을 가진다.
- 부서별 권한은 해당 부서 역할을 가진 사원에게만 적용된다.

## 시스템 권한
- EMPLOYEE: 일반권한
- DEPT_MANAGER: 부서 관리자 권한
- ADMIN: 시스템 총괄 권한

## 부서별 권한
- FRANCHISE : 프랜차이즈 관리
- IT : 시스템/자원 관리
- HR : 입사/퇴사/인사정보 (시스템/마스터 데이터 제외)
- FACILITY : 회의실/시설 관리
