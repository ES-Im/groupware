# 권한 규칙

## 시스템 권한
- EMPLOYEE: 일반권한
- DEPT_MANAGER: 부서 관리자 권한
- ADMIN: 시스템 총괄 권한

| 권한명                     | 설명                  |  EMPLOYEE  |  DEPT_MANAGER  |  ADMIN  |
|-------------------------|---------------------|:----------:|:--------------:|:-------:|
| approveRegister         | 가입 허가               |            |       o        |         |
| updateInfoBySelf        | 본인 정보 관리            |     o      |                |         |
| updateInfoByDeptManager | 같은 부서 직원의 제한된 정보 관리 |            |       ㅇ        |         |
| updateInfoByAdmin       | 모든 부서 내 사원 정보 수정    |            |                |    ㅇ    |
